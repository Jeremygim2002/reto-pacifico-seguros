# Reto Inmobiliario - Pacífico Seguros

## Descripción del Proyecto
Solución automatizada para el reto inmobiliario

## Estructura del Repositorio
* data/: Archivos de entrada crudos (activos.csv, gastos_ope.csv, contratos_alquiler.csv).
* src/: Código fuente en Python para la limpieza y consolidación de datos (procesamiento.py).
* output/: Base de datos final limpia exportada en formato Excel (Inmuebles_Listos.xlsx).

## Instrucciones de Ejecución
1. Clonar el repositorio en el entorno local.
2. Colocar los archivos de datos originales dentro de la carpeta data/.
3. Abrir la terminal en la raíz e instalar las dependencias usando: pip install -r requirements.txt
4. Ejecutar el script principal de procesamiento de datos con: python src/procesamiento.py
